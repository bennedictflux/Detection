# KK88Car - Premium Car Audio Systems

A modern, responsive e-commerce website for KK88Car, specializing in premium car audio systems including radios, speakers, and combo packages.

## 🚀 Features

### 🎨 Design & UI
- **Modern Orange & Black Theme**: Professional color scheme reflecting the brand identity
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Scroll-triggered animations and hover effects
- **Clean Typography**: Using Inter font for excellent readability

### 🛒 E-commerce Functionality
- **Shopping Cart**: Full cart functionality with add, remove, and quantity controls
- **Product Catalog**: Organized sections for radios, speakers, and combo packages
- **Combo Packages**: Special bundled deals with savings calculations
- **Local Storage**: Cart persistence across browser sessions

### 📱 User Experience
- **Smooth Scrolling Navigation**: Seamless section transitions
- **Mobile-Friendly**: Hamburger menu and touch-optimized interface
- **Contact Forms**: Working contact and newsletter subscription forms
- **Product Notifications**: Visual feedback when items are added to cart

### ⚡ Performance
- **Lazy Loading**: Images load as needed for better performance
- **Optimized Animations**: RequestAnimationFrame for smooth scrolling
- **Modern CSS**: CSS Grid and Flexbox for efficient layouts
